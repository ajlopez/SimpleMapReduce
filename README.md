# SimpleMapReduce

Simple MapReduce implementation, written in JavaScript. WIP.

## Installation

Via npm on Node:

```
npm install simplemapreduce
```


## Usage

Reference in your program:

```js
var simplemapreduce = require('simplemapreduce');
```
